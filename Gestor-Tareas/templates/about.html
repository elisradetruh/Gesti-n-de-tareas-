Acerca del Gestor de Tareas

Descripción

El Gestor de Tareas es una aplicación web simple desarrollada con Flask que te permite crear, gestionar y hacer seguimiento de tus tareas diarias. Es perfecta para organizar tu trabajo y mantener un control de lo que necesitas hacer.

Funcionalidades
  • Agregar tareas: Crea nuevas tareas fácilmente
  • Marcar como completada: Marca las tareas cuando las termines
  • Desmarcar tareas: Puedes desmarcar tareas si cambias de opinión
  • Eliminar tareas: Elimina tareas que ya no necesitas
  • Estadísticas: Ve el progreso de tus tareas
Tecnologías utilizadas
  • Flask - Framework web de Python
  • Bootstrap 5 - Framework CSS para el diseño
  • Bootstrap Icons - Iconografía
  • Jinja2 - Motor de plantillas
  • Python 3.x - Lenguaje de programación
